<p>تابعی که تابع دیگری را به عنوان ورودی می‌گیرد و یا تابعی را به عنوان خروجی بر می‌گرداند.</p>

```javascript
function unary(fn) {
	return function one(arg) {
		return fn(arg);
	}
}
```
