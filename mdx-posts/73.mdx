<p>متغیرها همانند سیم‌هایی هستن که به <a href="https://www.berneti.ir/72/%D8%A7%D9%86%D9%88%D8%A7%D8%B9-%D9%85%D9%82%D8%AF%D8%A7%D8%B1%D9%87%D8%A7-%DB%8C%D8%A7-values-%D8%AC%D8%A7%D9%88%D8%A7%D8%A7%D8%B3%DA%A9%D8%B1%DB%8C%D9%BE%D8%AA-%D8%A8%D9%87-%D8%B2%D8%A8%D8%A7%D9%86-%D8%B3%D8%A7%D8%AF%D9%87-%DB%B1">انواع مقدارها</a> متصل می‌شوند به مثال انتصاب 'Narwhal' به متغیر pet توجه کنید.</p>

```js
let pet = 'Narwhal';
```

<h4>عملیات انتصاب یا =</h4><p>در سمت چپ علامت = متغیر یا همان سیم ما با یک اسم مانند pet در مثال بالا قرار می‌گیرد و در قسمت راست یک مقدار ویا یک عبارت که حاصل اجرای آن عبارت توسط جاوااسکریپت محاسبه شده و در نهایت یک مقدار قرار می‌گیرد.</p><h4>خواندن مقدار یک متغیر</h4><p>در لحظه‌ی اجرای کد زیر مقدار متغیر pet محاسبه شده و برای متد log ارسال می‌شود.&nbsp;</p>

```js
console.log(pet);
```

<p>مقدار خروجی چاپ شده حاصل از اجرای کد زیر را شما حدس بزنید.</p>

```js
function double(x) {
  x = x * 2;
}

let money = 10;
double(money);
console.log(money); // ?
```

<p>سعی کنید یک مدل ذهنی برای قطعه کد زیر تصور کنید</p>

```js
let isSad = true;
let isHappy = !isSad; // The opposite
let isFeeling = isSad || isHappy; // Is at least one of them true?
let isConfusing = isSad && isHappy; // Are both true?
```

<p>آیا به درستی به نتیجه مشابه دست یافتید؟ باید توجه کنید که انواع مقادیر اولیه یا primitive در این مدل ذهنی ما منحصر به فرد هستن و صرفا یک مقدار true و یک مقدار false از نوع Boolean در دنیای جاوااسکریپت وجود دارد.</p>

<figure className="image image_resized" style={{"width":"53.75%"}}><img src="/img/js-26.png" /><figcaption>Just Javascript By <a href="https://illustrated.dev/">Maggie Appleton</a></figcaption></figure><p>حال اجازه دهید به تفاوت بین مقادیر دسته‌ی primitive و objects and functions بپردازیم. یکی دیگر از تفاوت‌های دسته‌ی دوم علاوه بر قابل تغییر بودن آنها این است که می‌توان مقادیر جدیدی از آن در دنیای جاوااسکریپت تولید کرد که صرفا مانند مقادیر دسته‌ی primitive ثابت و یکتا نیستند. به مثال‌های زیر توجه کنید.</p><p>سعی کنید یک مدل ذهنی از کدهای زیر که عملیات انتصاب یک مقدار از دسته‌ی مقادیر primitive است را تصور کنید.</p>

```js
let sisters = 3;
let musketeers = 3;
```

<figure className="image image_resized" style={{"width":"52.69%"}}><img src="/img/js-27.png" /></figure><p>حال مدل ذهنی انتصاب چند مقدار از نوع object که ما هر بار در هر خط با استفاده از {} یک نمونه جدید از آن را ایجاد کردیم مقایسه کنید.</p>

<figure className="image image_resized" style={{"width":"46.54%"}}><img src="/img/js-28.png" /></figure><p>سعی کنید حدس بزنید در هر کدام از &nbsp;۳ حلقه for زیر چه تعداد از نوع‌های متفاوت از مقادیر به متد console.log ارسال شد.</p>

```js
for (let i = 0; i < 7; i++) {
  console.log(2);
}
for (let i = 0; i < 7; i++) {
  console.log({});
}
for (let i = 0; i < 7; i++) {
  console.log(function() {});
}
```

<p>وقتی حلقه اول ۷ مرتبه اجرا می‌شود، ۷ بار تنها <strong>یک مقدار 2</strong> که از نوع primitive می‌باشد یعنی آن عدد 2 صرفا یک شیئ منحصربفرد در دنیای جاوااسکریپت است.</p><p>اما پس از اجرای حلقه دوم و سوم ۷ شیئ متفاوت و متمایز در دنیای جاوااسکریپت ایجاد شده و به متد console.log ارسال می‌شود.</p>
