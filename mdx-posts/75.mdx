<h4>نکته‌ی اول: object تو در تو وجود ندارد!</h4><>object تو در تو وجود ندارد و هر جفت علامت کروشه {} نشانه‌ی یک آبجکت متمایز و جدید است.</><h4>گام اول: تعریف متغیر sherlock<br />&nbsp;</h4>

```js
let sherlock = {
  surname: 'Holmes',
  address: { city: 'London' }
};
```

<h4>گام دوم: تعریف متغیر sherlock<br />&nbsp;</h4>

```js
let john = {
  surname: 'Watson',
  address: sherlock.address
};
```

<h4>نکته‌ی دوم: پراپرتی‌های هر شیئ همیشه به یک مقدار اشاره می‌کنن!</h4><>نکته‌ی دوم پراپرتی‌های هر شیئ همیشه به یک مقدار اشاره می‌کنن نه یک متغیر یا پراپرتی دیگر</>

<figure className="image image_resized" style={{"width":"54.74%"}}><img src="/img/js-14.png" /></figure><h4>گام سوم: ایجاد تغییر در پراپرتی‌ها</h4><>نتیجه کد زیر را با استفاده از مدل ذهنی گام قبلی حدس بزنید.</>

```js
john.surname = 'Lennon';
john.address.city = 'Malibu';
```

<>مدل ذهنی بصورت زیر تغییر می‌کند.</><figure className="image image_resized" style={{"width":"60.58%"}}><img src="/img/js-15.png" /></figure>

```javascript
console.log(sherlock.surname); // "Holmes"
console.log(sherlock.address.city); // "Malibu"
console.log(john.surname); // "Lennon"
console.log(john.address.city); // "Malibu"
```

<>نگاه دقیقتر به فرایند اعمال تغییرات</><figure className="image image_resized" style={{"width":"60.01%"}}><img src="/img/js-16.gif" /></figure>
<>همانطور که مشاهده می‌کنید با تغییر مقدار city آدرس john، مقدار city آدرس sherlock هم عوض می‌شود (صرفا پراپرتی city به یک مقدار دیگر اشاره می‌کند زیرا طبق آنچه آموختیم مقادیر Primitive غیر قابل تغییر هستند.)</><h4>راه حل اول ایجاد شیئ address جدید</h4>
<>مشکلی که ما در این مرحله با آن روبرو هستیم تغییر آدرس sherlock در زمان تغییر آدرس john است برای این کار با کد زیر می‌توانیم بصورت زیر یک object جدید برای پراپرتی address تعریف کنیم.</>

```js
// Replace Step 3 with this code:
john.surname = 'Lennon';
john.address = { city: 'Malibu' };
```

<>همانطور که مشاهده می‌کنید با عملیات انتصاب = به پراپرتی address این پراپرتی به یک آبجکت جدید که با خط نارنجی مشخص شده‌است متصل شد.</>
<h4>راه حل دوم هیچ تغییری در شیئ john ایجاد نکنیم</h4><>راه حل بعدی این است که برای اینکه متغیر john شامل مقادیر جدیدی باشد با عملیات انتصاب یک شیئ جدید ایجاد کرده و متغیر john را به آن متصل کنیم.</>

```js

  surname: 'Lennon',
  address: { city: 'Malibu' }
};
```
