<h4>تابع unary</h4><p>تابعی که یک ورودی می‌گیرد و یک مقدار به عنوان خروجی بر می‌گرداند.</p>

```javascript
// unary
function increment(x) {
	return sum(x,1);
}
```
 <h4>تابع باینری یا binary</h4><p>تابعی که دو ورودی می‌گیرد و یک مقدار به عنوان خروجی بر می‌گرداند.</p>

```javascript
// binary
function sum(x,y) {
	return x + y;
}
```

<h4>تابع ternary</h4><p>تابعی که سه ورودی می‌گیرد و یک مقدار به عنوان خروجی بر می‌گرداند.</p><h4>تابع n-ary</h4><p>تابعی که بیشتر از دو ورودی می‌گیرند(n ورودی)، توابعی از این دست کمتر متداول هستند زیرا هر چه یک تابع ورودی‌های بیشتری داشته باشه سخت‌تر می‌تواند با توابع دیگر کار کند با توجه به اینکه اکثر توابع در یک برنامه خوب بصورت unary یا binary پیاده‌سازی شده‌اند.</p><h4>تابع Variadic</h4><p>تابعی که تعداد ورودی‌های آن مشخص نیست.</p>

```javascript
function f(...args) {
	return args;
}
```
