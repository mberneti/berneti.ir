<p>آرگمان و پارامتر دو روی یک سکه هستند ولی اساسا با هم تفاوت دارند. آرگمان مقداری هست که زمان فراخوانی تابع به آن می‌دهیم که در نهایت به پارامتر‌هایی که زمان تعریف تابع مشخص شده مقداردهی می‌شوند.</p>
<p>در مثال زیر 3 و 4 آرگمان و a و b پارامتر‌های تابع sum هستند:</p>

```javascript
function sum(a,b) {
	return a + b;
}

sum(3,4);
```
